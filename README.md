# Python Expression Interpreter

Simple Math Interpreter in Python

## Using

```bash
python main.py
```

## Example

```bash
calc > 5
5.0
calc > -110
-110.0
calc > --110
110.0
calc > ---110
-110.0
calc > 5*4+5
25.0
calc > 5/2+4*4+(5/4-2)+1/1
18.75
```
